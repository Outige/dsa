This README tries to discuss the implementation of the python hash table (dict)

# Performance

| Operation      | Average Case  | Amortized Worst Case  |
|----------------|---------------|-----------------------|
|  k in d        | O(1)          | O(n)                  |
|  Copy[3]       | O(n)          | O(n)                  |
|  Get Item      | O(1)          | O(n)                  |
|  Set Item[1]   | O(1)          | O(n)                  |
|  Delete Item   | O(1)          | O(n)                  |
|  Iteration[3]  | O(n)          | O(n)                  |
